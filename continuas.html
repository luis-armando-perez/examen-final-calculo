Distribución Uniforme

En una distribución uniforme continua todos los valores dentro del intervalo [a, b] tienen la misma probabilidad.

Densidad:

f(x) = 1 / (b - a)

Es útil para modelar fenómenos con distribución constante dentro de un rango definido.

Distribución Normal

La distribución normal modela variables continuas que se concentran cerca de la media.

Su función de densidad es:

\[ f(x)=\frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

La probabilidad se calcula integrando esta curva. Con valores estándar usamos la tabla Z.

Distribución Exponencial

La distribución Exponencial modela el tiempo entre eventos. Se define por su parámetro λ (tasa).

Función de Densidad

f(x) = λ * e^(-λx), x ≥ 0

E[X] = 1/λ

Var(X) = 1/λ²

Distribución Log-Normal

La distribución Log-Normal es aquella donde el logaritmo de la variable sigue una distribución normal. Se define por los parámetros μ y σ.

Función de Densidad

f(x) = (1 / (xσ√2π)) * exp(-(ln(x) - μ)² / (2σ²)), x > 0

E[X] = exp(μ + σ²/2)

Var(X) = (exp(σ²) - 1) * exp(2μ + σ²)

Distribución Gamma

La distribución Gamma describe el tiempo hasta α eventos en un proceso de Poisson. Se define por los parámetros α (shape) y β (rate).

Función de Densidad

f(x) = (β^α / Γ(α)) * x^(α-1) * e^(-βx), x > 0

E[X] = α / β

Var(X) = α / β²

Distribución Beta

La distribución Beta es continua y definida en [0,1], utilizada para modelar proporciones y probabilidades. Sus parámetros α y β controlan la forma.

Función de Densidad

f(x) = x^(α-1) * (1-x)^(β-1) / B(α, β), 0 ≤ x ≤ 1

E[X] = α / (α + β)

Var(X) = αβ / ((α + β)² (α + β + 1))
